# POE-VEN-02 — Atención al Cliente
**Versión:** v1.0  
**Área:** Ventas  
**Responsable:** Asesor de Ventas  
**Aprobado por:** Gerente General  
**Fecha:** 13 de noviembre de 2025

## 1. Objetivo
Establecer el procedimiento estándar de atención al cliente para garantizar comunicación clara y servicio profesional.

## 2. Alcance
Aplica al Asesor de Ventas y al Administrador cuando dé soporte.

## 3. Documentos relacionados
- POE-VEN-01 Proceso de Ventas
- POL-VEN-01 Política de Tarifas

## 4. Procedimiento de atención
1. Responder mensajes y llamadas en máximo **15 minutos**.
2. Solicitar información básica:
   - Nombre del cliente  
   - DNI o RTN  
   - Tipo de servicio  
   - Punto de carga / entrega  
3. Confirmar disponibilidad de unidades.
4. Presentar tarifa oficial al cliente.
5. Registrar solicitud en sistema (Odoo).

## 5. Prohibiciones
- Dar tarifas no aprobadas.
- Prometer horarios imposibles.
- No registrar solicitudes.
