# MAN-LEG-01 — Plantillas de Contratos  
**Versión:** v1.0  
**Área:** Legal  
**Responsable:** Gerente General  
**Fecha:** 13 de noviembre de 2025  

---

## 1. Objetivo
Proveer plantillas base para contratos utilizados en TRALO.

---

## 2. Plantilla — Contrato de Operador
```
CONTRATO DE PRESTACIÓN DE SERVICIOS

Entre TRALO y _____________________________________ (Operador),
identificado con DNI ____________________________,
se acuerda la prestación de servicios de transporte bajo las siguientes condiciones:

1. El operador se compromete a cumplir políticas TRALO.  
2. Mantener buena conducta y comunicación.
3. Usar la unidad únicamente para fines autorizados.
4. Recibir pagos según tarifa acordada.

Firma Operador: __________________  
Firma TRALO: ______________________
```

---

## 3. Plantilla — Contrato de Proveedor
```
CONTRATO DE SUMINISTRO

TRALO contrata a _________________________________ (Proveedor),
RTN _____________________________________________,
para suministrar productos/servicios bajo las siguientes condiciones:

1. Cumplir calidad y plazos.
2. Presentar factura legal.
3. Precios acordados no pueden modificarse sin autorización.

Firma Proveedor: _______________  
Firma TRALO: ____________________
```

---

## 4. Plantilla — Contrato de Cliente
```
CONTRATO DE SERVICIO DE TRANSPORTE

Entre TRALO y ____________________________ (Cliente),
RTN ______________________________________,
se acuerda lo siguiente:

1. TRALO transportará mercancía en las rutas autorizadas.
2. Cliente debe pagar al recibir factura.
3. Cliente garantiza información correcta de carga.

Firma Cliente: _____________________  
Firma TRALO: _______________________
```
